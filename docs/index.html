Shorten Url Service

The main.py file contains the short_url_encode function, which takes a parameter url (either a string or an int) and outputs the result of short_url.encode_url from the short_url library.

Short URL is a Python implementation for generating Tiny URL- and bit.ly-like URLs.

To use the function: 

from main import short_url_encode

short_url_encode(1234567890)
short_url_encode('https://www.google.com')

Main

The main.py file serves as the entry point for the program. By calling python main.py in the terminal, the program will prompt the user for the input values and output the result. 


python main.py

Tests

The tests/sort_url_encode.py file contains the unit tests for the project. To run the tests, run python -m unittest tests/short_url_encode.py in the terminal.
